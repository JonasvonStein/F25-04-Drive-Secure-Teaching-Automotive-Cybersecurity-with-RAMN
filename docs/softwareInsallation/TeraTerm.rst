TeraTerm Installation Guide
===========================

TeraTerm is used to connect with the RAMN over USB

1. Go to Tera Term's `website <https://teratermproject.github.io/index-en.html>`_ and follow instructions for installation.
2. Go to Tera Term's `github <https://github.com/TeraTermProject/teraterm/releases>`_.
3. Install the .exe file that will work with your processor.
